# GreenLeaf-Sales-Dashboard-Tableau



The GreenLeaf Sales Dashboard is an interactive Tableau dashboard designed to analyze sales performance and provide actionable business insights. The dashboard enables users to monitor key sales metrics, identify top-performing products, evaluate regional performance, and track sales trends through interactive visualizations.

Live Dashboard:
View Dashboard on Tableau Public

🎯 Objectives
Monitor overall sales performance.
Analyze revenue and order trends over time.
Evaluate product category performance.
Compare sales across territories and regions.
Support data-driven decision-making through interactive analytics.
